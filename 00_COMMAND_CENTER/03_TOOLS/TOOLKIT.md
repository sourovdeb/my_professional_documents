# Toolkit — free & open-source tools to write, design, and live easier

*All free or open-source. All chosen to work the way you already work: plain text, local files, low overhead, ADHD-friendly. Recycle these; don't go shopping for more.*

The principle: **fewer tools, deeply used.** Every new app is a new place to lose things. Pick one per job and stop.

---

## Write

| Tool | Why it fits you | Cost |
|---|---|---|
| **Obsidian** | One vault of plain `.md` files = your whole second brain, ideas, drafts. Matches this repo exactly. Free for personal use; files stay yours forever. | Free |
| **Zettlr** | Fully open-source Markdown editor if you prefer FOSS; great for long pieces, citations, exports to PDF/Word. | Free (GPL) |
| **LanguageTool** | Grammar + style checker (FR + EN). Run it **last**, on edit day. Self-hostable; free tier is plenty. | Free |
| **Hemingway-style check** | Read aloud is your best editor; for a tool, LanguageTool's clarity hints do the job. | Free |
| **Git + this repo** | Version history = the Forensic Auditor's instinct applied to your own writing. Nothing is ever lost. | Free |

## Design (CV, blog images, simple branding)

| Tool | Use | Cost |
|---|---|---|
| **Penpot** | Open-source design/prototyping (Figma alternative). Blog headers, social cards, a clean personal brand. Browser-based, no install. | Free |
| **GIMP** / **Krita** | Image editing / illustration if you ever need raster work. | Free |
| **Inkscape** | Vector logos, simple diagrams, print-ready PDFs. | Free |
| **Unsplash / Pexels** | Free, license-clear photos for blog posts. Always check the license line. | Free |
| **Google Fonts** | License-clear typefaces for CV and site. | Free |

## Capture, plan, remember (the ADHD external brain)

| Tool | Use | Cost |
|---|---|---|
| **Obsidian / Logseq** | One capture inbox; one board. (Pick one. Logseq is fully FOSS, outliner-style — good for daily logs.) | Free |
| **Syncthing** | Sync your notes across devices without a cloud account or subscription. Private. | Free |
| **systemd --user timers** (Linux) / phone alarms | Fire your meds prompt and wind-down prompt *for* you. The machine remembers so you don't. | Free |
| **A 25-minute timer** | Any timer. The timer is the boss. | Free |

## Comfort & focus (make the day easier on a hard brain)

- **f.lux / Night Light** — warm the screen at night; protects the sleep that protects your mood.
- **Cold Turkey / LeechBlock** (free tiers) — block the doomscroll during a writing block. Remove friction by removing temptation.
- **Speech-to-text** — your phone's built-in dictation. On a low-energy day, *talk* the draft, clean it later. You speak more freely than you type.
- **Noise** — a single playlist or brown-noise tab you always use signals "now we write." Same cue every time = faster start.
- **Dark mode + big font everywhere** — reduce the friction that an overstimulated nervous system reads as "too hard."

## The whole stack, in one line

**Obsidian for words → Git/this repo for history → LanguageTool to polish → Penpot for visuals → Syncthing to sync → timers to remember.** Six tools, all free, all yours. Set it up once on a good day, then stop fiddling and just write.
