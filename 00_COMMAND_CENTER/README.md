## The weekly rhythm (the whole system on one screen)

This is deliberately tiny. A plan you can't run on a crash day is the wrong plan.

- **Every day:** the four anchors (see `08_SELF_CARE/`). If all four happen, the day counts.
- **Most days:** one 500-word draft into `01_WRITE/drafts/` — 25 minutes, timer on, no editing. (See the daily workflow.)
- **One stable day a week:** edit 2–3 drafts, schedule them, run one automation batch (jobs or outreach), empty the idea inbox.
- **Once a month:** psychiatric review; re-read this README; delete what you've outgrown.

You do not need to do all of this. On a low day, do the anchors and one line of capture. That *is* the system working.

## Working rules (so future-you and any AI agent stay on track)

1. **Active voice. Human. Never monotone.** Write like you talk to a friend who respects you.
2. **One place, not five apps.** This repo is the place. Resist the urge to start a new system.
3. **Every new creation gets its own branch.** You copy-paste to publish; nothing auto-publishes.
4. **Official and legal sources only** for facts about your condition, rights, and money.
5. **Decide the system once, on a good day. Then stop redesigning it** and just use it.
