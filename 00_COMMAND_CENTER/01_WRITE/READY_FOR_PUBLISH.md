## How to Publish

**To WordPress (your home base):**
1. Open the draft in `drafts/`
2. Copy the body text (everything below the `---` divider)
3. WordPress → Posts → Add New
4. Paste into the block editor
5. Set **Title**, **Slug** (use the one above), **Tags**, **Category**
6. Click **Save as Draft** → **Preview** → **Schedule** for the date you want
7. Close the draft file, delete from `drafts/`, mark as published somewhere you'll remember

**To Medium (1–2 days after WordPress):**
1. Open the essay again
2. Medium → Write a story
3. Paste the body
4. Set title, tags (use the same ones from WordPress)
5. Add this canonical link in the editor: `https://sourovdeb.com/[slug]`
6. Publish
