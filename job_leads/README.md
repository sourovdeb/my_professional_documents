# 💼 Job Leads Tracker

Track every opportunity here. Used by:
- `automation_scripts/indeed_scraper.py` - Auto-populates
- `wordpress_integration/wp_publisher.py` - Link to essays about job search
- Your weekly review process

## Files

### `indeed_leads.json`
Auto-generated by Indeed scraper. Contains:
- Job title, company, location, salary
- Description, URL
- Posted date
- **status**: new | interesting | applied | rejected | archived
- **applied**: true/false (when you formally apply)

### `applications.json`
**Manual:** Track formal applications you submit

```json
{
  "applications": [
    {
      "job_id": "indeed_12345",
      "company": "ACME Corp",
      "title": "English Teacher",
      "applied_date": "2026-06-02",
      "cover_letter": "path/to/cover_letter.md",
      "status": "awaiting_response",
      "response_date": null,
      "notes": "Phone screen scheduled 2026-06-10"
    }
  ]
}
```

## Weekly Workflow

1. **Monday morning:** Review `indeed_leads.json`
2. **Tuesday:** Pick 2-3 marked "interesting"
3. **Wednesday:** Write tailored cover letter
4. **Thursday:** Submit application + add to `applications.json`
5. **Sunday:** Follow up on awaiting responses

## Tips

- Mark status regularly (not just "applied")
- Write notes about each company (why you're interested)
- Keep cover letters for future reference
- Track rejection reasons (helps refine future apps)

## Statistics (Track Progress)

```bash
# Count total jobs found this month
jq 'length' job_leads/indeed_leads.json

# Count jobs you've applied to
jq '.applications | length' job_leads/applications.json

# See application success rate
jq '.applications | map(select(.status=="offer")) | length' job_leads/applications.json
```
