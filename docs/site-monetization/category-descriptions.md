# Category Descriptions

> **Where this goes:** WP → Posts → **Categories** → hover a category → **Edit** → paste into the **Description** field → Update. Steps in `implementation-manual.md` §7.
>
> Why it matters: unique category descriptions (a) tell Google what each section is about, (b) show real text on category archive pages instead of the generic default, and (c) help AdSense reviewers see coherent, organised content. Each is kept to 1–3 sentences with natural keywords.

---

## If a category isn't listed here

Use this fill-in formula so every category stays consistent:

```
[What the section covers] for [who it's for]. [What makes it useful / what the reader gets]. [Optional: that it's free / level-tagged / beginner-friendly].
```

Keep it to 1–3 sentences. Include the category's main keyword naturally in the first sentence.
