# Homepage Meta Tags + TLDR

> **Where this goes:** WP Dashboard → your SEO plugin (Yoast SEO or Rank Math) → Search Appearance / Titles → **Homepage**. Paste the title into the SEO Title field and the description into the Meta Description field. Steps in `implementation-manual.md` §4.
>
> Google truncates around **60 characters for titles** and **155–160 for descriptions** — the versions below are within limits. Pick ONE option from each set.

---

## 1. Homepage SEO Title (title tag)

Aim: lead with your strongest keywords (English teaching / ELT), include your name for brand.

**Option A — recommended (58 chars):**
```
Sourov DEB — English Teaching, ELT Methodology & Photography
```

**Option B — teaching-first (55 chars):**
```
Free ELT Lessons & English Teaching Resources | Sourov DEB
```

**Option C — authority angle (57 chars):**
```
Sourov DEB — CELTA-Certified English Trainer & Writer
```

---

## 2. Homepage Meta Description

Aim: one clear sentence on who it's for + what they get + that it's free. Include a soft call to action.

**Option A — recommended (152 chars):**
```
Free English teaching resources, ELT methodology, and photography guides by Sourov DEB, a CELTA-certified trainer. Practical lessons for learners and teachers.
```

**Option B — learner-focused (150 chars):**
```
Learn English with free, level-tagged lessons on grammar, vocabulary, and methodology — plus photography and AI guides. Clear, practical, and always free.
```

**Option C — broad personal brand (148 chars):**
```
Essays, English teaching lessons, photography, and philosophy by Sourov DEB. Free, practical content for curious learners and fellow teachers.
```

---

## 3. TLDR / homepage summary block (optional on-page text)

Some SEO plugins and themes let you add a short intro block on the homepage. A visible TLDR helps both readers and Google understand the site instantly. Drop this above the post feed:

> **TLDR:** This is the personal site of **Sourov DEB** — a CELTA-certified English trainer based in La Réunion. You'll find **free English teaching lessons** (grammar, vocabulary, methodology, level-tagged A1–C2), plus writing on **photography, AI tools, philosophy, and mental health**. Everything here is free to read — ads simply cover the running costs.

## 4. Open Graph / social preview (bonus)

If your SEO plugin has a Social tab, set these so links shared on Facebook/LinkedIn/X look clean:

- **OG Title:** `Sourov DEB — Free English Teaching & More`
- **OG Description:** `CELTA-certified ELT lessons, photography, and essays. Free to read.`
- **OG Image:** upload a 1200×630px banner (your logo/photo + tagline). This fixes the "blank preview" problem noted in the theme audit.

---

## 5. Per-post reminder

For each post going forward, the SEO plugin gives you the same two fields (title + description). Quick formula for ELT posts:

```
Title:        [Topic] — [Skill] (Level)      e.g. "Modal Verbs for Advice — Grammar (B1)"
Description:  One sentence: what the learner will be able to do after reading, + level.
```
