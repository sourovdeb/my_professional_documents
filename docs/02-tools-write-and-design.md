# Tools to Write, Design, and Make the Work Comfortable

All free or open-source. The rule: a tool earns its place only if it removes
friction. If it adds steps, drop it. You already have everything you need to
start — this is for when a real need appears.

## Writing (where the words happen)

| Tool | Use it for | Link |
|------|-----------|------|
| **Markdown + this repo** | Your source of truth. Plain text, version-controlled, yours forever. | (already set up) |
| **Obsidian** | Linking ideas, daily notes, the topic bank — free for personal use. | https://obsidian.md |
| **LibreOffice Writer** | When you need a real word processor. | https://www.libreoffice.org |
| **LanguageTool** | Grammar/style checker, open-source, self-hostable. | https://languagetool.org |
| **Hemingway-style editing** | Read aloud + cut. The best editor is your own ear. | — |

**Keep drafting and editing separate.** Drafting is generous; editing is ruthless.
Doing both at once is why the page feels heavy.

## Design (so the work looks like it matters)

| Tool | Use it for | Link |
|------|-----------|------|
| **GIMP** | Image editing, open-source Photoshop. | https://www.gimp.org |
| **Inkscape** | Vector graphics, logos, simple diagrams. | https://inkscape.org |
| **Canva (free)** | Fast social/blog headers without the learning curve. | https://www.canva.com |
| **Unsplash / Pexels** | Free, license-clear photos for posts. | https://unsplash.com · https://pexels.com |
| **Excalidraw** | Hand-drawn-style diagrams in seconds. | https://excalidraw.com |
| **Google Fonts** | Free, open-licensed typefaces. | https://fonts.google.com |

A clean post needs only: one good header image, readable type, generous spacing.
WordPress themes handle this for you — don't over-design.

## Comfort (so writing doesn't cost you health)

This is not optional for you — it is infrastructure. See
[docs/06-self-care-and-your-condition.md](06-self-care-and-your-condition.md).

| Need | Free tool | Link |
|------|-----------|------|
| Rest your eyes / reduce strain | **f.lux** (warm screen at night) | https://justgetflux.com |
| Write by voice on low-energy days | **OS dictation** (built into Windows/Mac/Android/iOS) | — |
| Read drafts back to you | **OS text-to-speech** or **Read Aloud** browser ext. | — |
| Focus without pressure | **Pomofocus** (timer, no signup) | https://pomofocus.io |
| Reduce typing strain | **Espanso** (text expander, open-source) | https://espanso.org |

## The minimalist stack (start here, add nothing else yet)

1. This repo + Markdown for writing.
2. WordPress for publishing.
3. Your ear (read aloud) for editing.
4. One timer for pacing.

Everything above is there *if a need appears.* Don't collect tools. Collect
finished pieces.
