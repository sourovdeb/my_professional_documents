# 3 — Tools: free & open-source, to write, design, and live easier

Everything here is free. Most is open-source. None of it locks you in. Recycle
the tools you already have before adding new ones.

## Write
- **Obsidian** (free) — markdown notes that link together; your idea bank and
  drafts in one place. Plays perfectly with this git repo.
- **LanguageTool** (open-source) — grammar/style checker, far more private than
  big-tech tools; self-hostable. https://languagetool.org
- **VS Code** (free) — you already use this repo in it; write markdown here and
  branch each draft (guide 2).
- **Zen / focus writing:** Obsidian's "focus mode" or `ghostwriter` (OSS) for
  distraction-free drafting.

## Design (covers, social cards, simple graphics)
- **Canva** (free tier) — fastest for blog headers and LinkedIn/Substack cards.
- **GIMP** (open-source) — full image editor when you need real control.
- **Inkscape** (open-source) — logos, vector graphics, clean diagrams.
- **Excalidraw** (open-source) — hand-drawn-style diagrams for explaining ideas.
- **Unsplash / Pexels** — free, license-clear photos for posts.

## Publish & automate (free, official)
- **WordPress** + **Jetpack** (free) — auto-share new posts to socials.
- **n8n** (open-source) — visual automation; self-host free. Your job/email
  pipelines live here (guide 4).
- **Indeed / Gmail / Google Drive** — you already have these connected to this
  assistant; use them, don't rebuild them.

## Comfort / accessibility (matched to ADHD + low-energy days)
- **Text-to-speech:** built-in on your OS, or **NaturalReader** free tier — hear
  your draft read back to catch clunky sentences without re-reading.
- **Speech-to-text:** dictate drafts on hard days. OS dictation is free; the
  open-source **Whisper** is excellent if you want local, private transcription.
- **Pomodoro timer:** any free one. 15-min write, 5-min rest fits your loop.
- **Focus / blocking:** **LeechBlock** (open-source browser add-on) to mute
  socials while you write.
- **Reading ease:** dyslexia-friendly fonts, dark mode, larger line spacing —
  all free in Obsidian/VS Code settings.

## Rule for adding tools
Add a tool only when a *specific* friction hurts twice. A new app is not
progress; a finished post is. Quality over quantity applies to tools too.
