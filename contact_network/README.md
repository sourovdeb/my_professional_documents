# 👥 Contact Network Management

Build and maintain a network of writers, collaborators, recruiters, and mentors.

## Goals

✅ Find like-minded writers  
✅ Track outreach efforts  
✅ Build real relationships  
✅ Create collaboration opportunities  
✅ Get feedback on your work  

## Files Structure

### `potential_contacts.json`
Auto-generated by `contact_finder.py`. Format:
```json
{
  "name": "Jane Writer",
  "platform": "GitHub",
  "profile": "https://github.com/jane",
  "bio": "Writing about disability and tech",
  "followers": 1200,
  "found_date": "2026-06-02",
  "status": "not_contacted",
  "potential_email": "jane@gmail.com"
}
```

**Statuses:**
- `not_contacted` — Found but haven't reached out
- `outreach_sent` — Email sent, waiting for response
- `conversing` — Had exchange, ongoing
- `collaborating` — Active collaboration
- `keep_for_future` — Interesting but not right now

### `outreach_log.json`
**Manual:** Track your outreach efforts

```json
{
  "outreach": [
    {
      "contact_name": "Jane Writer",
      "platform": "GitHub",
      "outreach_date": "2026-06-02",
      "email_sent": true,
      "email_subject": "Collaboration on disability writing",
      "response": true,
      "response_date": "2026-06-03",
      "response_summary": "Interested in guest post exchange",
      "next_step": "Send guest post outline",
      "notes": "Very responsive, similar audience"
    }
  ]
}
```

### `collaborations_active.json`
Projects you're working on together

```json
{
  "collaborations": [
    {
      "title": "Disability Writing Series",
      "partner": "Jane Writer",
      "started": "2026-06-03",
      "status": "outline_stage",
      "goal": "Co-author 4-part series on disability in tech",
      "deadline": "2026-08-01",
      "last_update": "2026-06-02",
      "link_to_essays": "daily_essays/collab_disability_tech_pt1.md"
    }
  ]
}
```

## Weekly Contact Workflow

### Monday: Review Potential Contacts
```bash
# See who you found this week
tail -20 contact_network/potential_contacts.json

# Mark interesting ones:
# 1. Copy profile URL
# 2. Read their latest work
# 3. Note why they're interesting
```

### Wednesday: Outreach Batch
```bash
# Select 3-5 contacts to reach out to
# Generate personalized email templates
python3 automation_scripts/contact_finder.py \
  --generate-email <contact_name>

# Send email manually (Gmail)
# Then log in outreach_log.json
```

### Friday: Follow-up Check
```bash
# Check responses in email
# Update status in potential_contacts.json
# Log in outreach_log.json
```

### Sunday: Collaboration Review
- Check progress on active collaborations
- Plan next steps
- Update collaborations_active.json

## Email Outreach Templates

### First Outreach (Cold)
```
Subject: [Collaboration] Writing about disability & [their topic]

Hi [Name],

I came across your recent article on [SPECIFIC WORK].
Your perspective on [SPECIFIC POINT] really resonated with me.

I'm writing about disability in [YOUR FIELD], and I think our approaches align.

Would you be open to:
- Guest post exchange on our blogs?
- Collaboration on a specific topic?
- Just sharing resources/insights?

My work: https://www.sourovdeb.com
Latest: [LINK TO RECENT ESSAY]

Looking forward to connecting,
Sourov
```

### Follow-up (If no response after 1 week)
```
Subject: RE: [Collaboration] Writing about disability & [their topic]

Hi [Name],

Just checking if my previous email landed (sometimes things get buried).

No pressure—I genuinely just appreciate your work and thought we might find common ground.

If you're interested, I'd love to connect.

Best,
Sourov
```

### Response to Their Email
```
Thanks so much for getting back to me! 

I really appreciated [SPECIFIC THING THEY SAID].

For the collaboration, I was thinking we could:
[SPECIFIC, CONCRETE IDEA]

What do you think? Happy to adjust based on your bandwidth.

Looking forward,
Sourov
```

## Tips for Building Relationships

1. **Read their work first.** Not just headlines—actual deep read.
2. **Be specific.** Don't just say "I like your writing." Say *why*.
3. **Offer value.** What can you bring to them? Not just "let's collaborate."
4. **Be authentic.** Real connections > network collection.
5. **Follow up.** One email isn't enough. But don't be creepy.
6. **Share their work.** Actually promote contacts you care about.
7. **Introduce them.** Connect contacts with other contacts they'd like.

## Red Flags (Avoid)

❌ Generic outreach ("I love your blog!")  
❌ Immediately asking for favors  
❌ No genuine interest in their work  
❌ Spammy follow-ups  
❌ Taking but not giving  

## Success Metrics

Track these monthly:
- Total contacts found: `jq 'length' contact_network/potential_contacts.json`
- Outreach emails sent: `jq '.outreach | length' contact_network/outreach_log.json`
- Response rate: `jq '.outreach | map(select(.response==true)) | length' contact_network/outreach_log.json`
- Active collaborations: `jq '.collaborations | length' contact_network/collaborations_active.json`

**Goal:** 3-5 quality relationships per month (not 50 weak connections)

## Sample Contact Journey

```
Week 1: Find Jane Writer via GitHub
        Read 3 of her recent essays
        Note: "Writes about disability + tech, 1.2k followers"

Week 2: Send personalized email about collaboration
        Reference specific article
        Offer concrete idea

Week 3: Jane responds! Interested in guest exchange
        Add to outreach_log.json with response
        Plan first guest post outline

Week 4: Exchange detailed outline via email
        Agree on timeline
        Add to collaborations_active.json

Month 2: Publish first joint essay to both blogs
        Promote each other's work
        Discuss next collaboration

Result: Long-term collaborator + cross-promotion opportunity
```

## Tools You Can Use

**Manual (Free):**
- Gmail + Spreadsheet to track
- GitHub for finding writers
- Twitter advanced search for finding your audience

**Automated:**
- HubSpot CRM (free) - track conversations
- Airtable (free) - dashboard view
- Hunter.io - find email addresses

**No tools needed:**
- Simple JSON files (you have them)
- Spreadsheet (Google Sheets)
- Notes app (Notion, Obsidian)

## Get Started Today

```bash
# 1. Find potential collaborators
python3 automation_scripts/contact_finder.py --search "disability writing"

# 2. Create outreach_log.json if missing
echo '{"outreach": []}' > contact_network/outreach_log.json

# 3. Create collaborations file
echo '{"collaborations": []}' > contact_network/collaborations_active.json

# 4. Pick 3 people you found
# 5. Read their latest work (30 mins)
# 6. Send personalized email (15 mins)
# 7. Log it in outreach_log.json
```

**Remember:** Quality over quantity. 3 real collaborators > 50 ignored follows.
