## How to Use These Sources

1. **Always link directly to official government sources** in your posts
2. **Check dates** — ensure guidance is current (note publication/update date)
3. **Verify with multiple sources** before publishing (cross-check rules)
4. **Note any pending changes** — if a rule changes on a specific date, mention it
5. **Distinguish between guidance and law** — what's mandatory vs. recommended
6. **Track new announcements** — set alerts on government websites

---

## Source Verification Checklist

Before citing any source:

- [ ] Source is official government website (not secondary commentary)
- [ ] Information is current (check last update date)
- [ ] Source explicitly addresses the topic (not speculation)
- [ ] Cross-check with 1–2 other official sources
- [ ] Note any conditions or exceptions mentioned in source
- [ ] Source is in English or has official English translation
- [ ] URL is working and permanent (not a temporary page)
