# AI Term Lessons — Index

Short (2–5 minute) educational lessons that explain one core AI term at a time, in
plain language, using everyday software analogies (Office, browsers, basic computer
tasks) and **Mistral Studio** as the running concrete example.

Each lesson ships as a matched pair:
- `NN_<Term>_Lesson_Script.md` — read-aloud script / outline, timed for video or as a
  written lesson.
- `NN_<Term>_Lesson.pptx` — a 6-slide companion deck.
