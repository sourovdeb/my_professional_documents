# Systematic Search Protocol

## Overview
For each initiative idea, conduct these 5 systematic searches in sequence.

## 1. Regulatory Search (Immediate Priority)
**Query Format**: "[Idea name] legal requirements [country/region] site:gov"
**Example**: "community radio license requirements India site:gov.in"
**Sources**: Official government websites, regulatory bodies
**Purpose**: Identify all legal/regulatory barriers before investment

## 2. Competitive Search (Week 1)
**Query Format**: "[Idea category] existing projects OR platforms"
**Example**: "Bengali audiobook platforms existing"
**Sources**: Google, GitHub, Crunchbase, NGO databases
**Purpose**: Avoid duplication, identify partners, understand market gaps

## 3. Funding Search (Week 2)
**Query Format**: "grants for [idea category] [region] 2024 OR 2025"
**Example**: "grants for digital media West Bengal 2025"
**Sources**: Grant databases, EU Funding Portal, Foundation directories
**Purpose**: Secure sustainable financing

## 4. Technical Search (Week 3)
**Query Format**: "how to build [technical component] tutorial"
**Example**: "how to build Chrome extension for scraping tutorial"
**Sources**: GitHub repos, Stack Overflow, official documentation
**Purpose**: Identify implementation approach and required skills

## 5. Partnership Search (Week 4)
**Query Format**: "NGOs working on [problem domain] [region]"
**Example**: "NGOs working on migration rights Greece"
**Sources**: NGO directories, LinkedIn, university partnerships
**Purpose**: Build network for implementation and validation

## Source Priority Hierarchy
1. **Primary**: Official government/regulatory sites
2. **Secondary**: Academic databases (Google Scholar, JSTOR)
3. **Tertiary**: NGO reports, established media
4. **Quaternary**: Social media, forums (MUST verify with primary sources)
