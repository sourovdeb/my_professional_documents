# Bengali Media Ideas (22-25)

## Idea 22: Village Story Pods
- **What**: Mobile recording booths touring rural West Bengal to capture oral histories
- **How**: Partner with local NGOs, use Raspberry Pi + USB mics, solar-powered
- **Where**: Villages in Nadia, Murshidabad, Birbhum
- **Cost**: $1,200
- **Risk**: Equipment theft/weather damage
- **Reversibility**: Reversible
- **Label**: DOCUMENTED
- **Status**: Validation phase (Weeks 1-4)

### Validation Tasks
- [ ] Identify 3 pilot villages
- [ ] Recruit 5 local storytellers
- [ ] Test recording equipment
- [ ] Upload first 20 stories to SoundCloud
- [ ] Collect user feedback

### Success Criteria
- 20 stories collected
- 80% positive feedback from listeners
- Equipment survives field conditions

## Idea 23: Bengali Audiobook Co-op
- **What**: Crowdsourced audiobook creation for public domain Bengali literature
- **How**: Volunteer narrators, Audacity, archive on Internet Archive
- **Where**: Global Bengali diaspora
- **Cost**: $800
- **Risk**: Copyright verification for public domain works
- **Reversibility**: Reversible
- **Label**: INFERRED
- **Status**: Not started

### Implementation Plan
1. Identify public domain Bengali works
2. Recruit volunteer narrators
3. Set up quality standards
4. Create distribution channels
5. Promote to diaspora communities

## Idea 24: Radio Drama for Social Change
- **What**: Weekly serialized dramas addressing taboo topics (mental health, domestic violence)
- **How**: Partner with local theatre groups, use community radio slots
- **Where**: West Bengal + Tripura
- **Cost**: $2,500
- **Risk**: Censorship from conservative groups
- **Reversibility**: Costly
- **Label**: DOCUMENTED
- **Status**: Pilot phase (Weeks 5-12)

### Pilot Plan
- 4 episodes, 30 minutes each
- Broadcast on 2 community radio stations
- Online streaming via website
- Target: 5,000 listeners, 70% engagement

## Idea 25: Folk Music Licensing Collective
- **What**: Platform for indie Bengali musicians to license tracks to radio/TV
- **How**: Blockchain-based smart contracts for royalties, WordPress + WooCommerce
- **Where**: Online, targeting Bengali media producers
- **Cost**: $3,500
- **Risk**: Legal complexity of music licensing
- **Reversibility**: Reversible
- **Label**: DOCUMENTED
- **Status**: Not started

### Technical Requirements
- User registration system
- Music upload and metadata tagging
- Smart contract integration
- Payment processing
- Royalties tracking
