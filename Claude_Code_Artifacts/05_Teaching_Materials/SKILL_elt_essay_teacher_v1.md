# SKILL: ELT Essay Teacher
**Version:** 1.0 | **Created:** 2026-06-14 | **Author:** Sourov DEB
**Scope:** Teach a learner to write English essays / extended writing — exam essays (IELTS Task 2, Cambridge B2/C1/C2, TOEIC), academic (EAP), or general argumentative/discursive — to the rubric they are actually assessed on. Not memoir/blog writing (see SKILL_portfolio_to_essays for that).
**Authority note:** This skill encodes the published assessment criteria and writing-pedagogy literature cited under SOURCES. It is not a guarantee of a band score; it is a teaching scaffold.

---

## OVERVIEW
Most essay teaching fails one way: it drills vocabulary and grammar while ignoring **Task Response** and **organisation**, which together carry half the marks on every major exam. This skill forces teaching to the *whole* rubric, in priority order, with criterion-referenced feedback and a redraft loop.

---

## TRIGGER CONDITIONS
Activate when the user wants to:
- Teach essay / opinion / discursive / academic writing in English.
- Prepare a learner for IELTS Writing Task 2, a Cambridge Main Suite writing paper, or an EAP essay.
- Mark a learner's essay and give actionable, level-appropriate feedback.
- Build a writing-development plan or a lesson on a writing sub-skill (cohesion, paragraphing, task analysis).

## MODULE 1 — DIAGNOSE (before teaching anything)
```
1. Which assessment? IELTS T2 / Cambridge (B2 First, C1 Advanced, C2 Proficiency) /
   TOEIC Writing / EAP / general. The rubric differs — teach to the right one (Module 2).
2. Learner level (CEFR A1–C2). Below B1, do not teach the 4-paragraph essay yet —
   build sentence and paragraph control first.
3. Get a BASELINE: one timed sample under exam conditions. Never plan from self-report.
4. Identify the task TYPE (for IELTS): opinion (agree/disagree), discussion (both views),
   problem–solution, advantages–disadvantages, two-part/direct question.
   Wrong essay type for the prompt caps the score regardless of language.
```

## MODULE 2 — THE ASSESSMENT MODEL (teach to this, not to "good writing")
```
IELTS WRITING TASK 2 — four criteria, 25% each (Cambridge/British Council/IDP, public descriptors):
  TR  Task Response       — addresses ALL parts; clear position; ideas extended + supported
  CC  Coherence & Cohesion — logical progression; paragraphing; referencing/substitution;
                              linkers used naturally (mechanical/over-use is penalised)
  LR  Lexical Resource    — range, precision, collocation, register
  GRA Grammatical Range & Accuracy — variety of structures + control
  KEY FACT: TR + CC = 50%. A grammatically clean essay that misses the task will not
  exceed Band 6. Teach in this priority: TR → CC → LR → GRA.

CAMBRIDGE MAIN SUITE (B2/C1/C2) — four subscales:
  Content · Communicative Achievement · Organisation · Language.
  "Communicative Achievement" = right register + genre conventions for the task.

EAP ESSAY — argument quality, evidence, source use/citation, academic register, cohesion.
```
Do not reproduce official descriptor tables (copyrighted). Teach the criteria in plain language; point the learner to the official public PDF (SOURCES).

## MODULE 3 — APPROACH SELECTION
```
PRODUCT / GENRE  — explicit models, target structures, exam "moves". Best for exam deadlines
                   and lower levels. (Genre approach: Hyland.)
PROCESS          — plan → draft → feedback → redraft; develops ideas + autonomy. Best for
                   fluency, EAP, and learners with time.
PROCESS–GENRE    — combine: model the genre, then run drafting cycles (Badger & White).
Default for exam prep: process–genre. Use a model essay, then redraft against criteria.
```

## MODULE 4 — ESSAY ARCHITECTURE (the moves)
```
INTRODUCTION   paraphrase the prompt + state position + signpost the essay (no memorised filler).
BODY ¶ (×2)    one controlling idea each. Pattern PEEL/TEEL:
                 Point (topic sentence) → Explain → Example/Evidence → Link back to question.
CONCLUSION     restate position + summarise main reasons. No new ideas.
PARAGRAPH RULE one idea per paragraph; topic sentence first; everything supports it.
LENGTH         meet the minimum (IELTS T2 ≥250 words) — under-length is auto-penalised;
                padding to hit length lowers TR.
```

## MODULE 5 — THE INVISIBLE 50% (where learners lose marks)
```
TASK RESPONSE
  - Answer EVERY part of the prompt. Underline task parts; check each is covered.
  - State a clear, consistent position; don't drift or sit on the fence (unless the task allows).
  - Extend ideas: claim → because → for example → therefore. Thin/unsupported ideas cap TR.
COHESION
  - Reference (this, such, the former) and substitution to avoid repetition.
  - Linkers signal logic — but over-using "Moreover/Furthermore/In conclusion" and memorised
    connectors is explicitly penalised. Cohesion should "attract no attention" at the top band.
  - Paragraphing IS cohesion; an unparagraphed essay is capped.
```

## MODULE 6 — FEEDBACK METHOD (criterion-referenced + redraft)
```
1. Score each criterion separately against Module 2. Name the band-limiting issue per criterion.
2. Do NOT correct every error. Prioritise the one error that most limits the band.
3. Use a CORRECTION CODE, not direct rewriting (learner self-corrects):
     sp=spelling  gr=grammar  ww=wrong word  wo=word order  prep=preposition
     T=tense  art=article  coll=collocation  ¶=new paragraph  reg=register  ?=unclear
4. REFORMULATE one or two sentences (write the learner's idea in stronger English) as a model.
5. Give ONE actionable target per criterion (e.g. TR: "add an example to ¶2";
     CC: "start ¶3 with a topic sentence"; LR: "replace 'good/bad' with precise words").
6. Redraft loop: learner rewrites addressing targets; remark. Improvement comes from the cycle.
```

## MODULE 7 — KNOWN CEILINGS (flag early)
```
- Off-topic / partial answer → Band ≤6 (IELTS) however strong the language.
- Memorised phrases / template overuse → penalised; reads as non-spontaneous.
- No paragraphing → CC capped.
- Under word count → auto-penalty.
- Translation-from-L1 sentence structure → GRA/LR ceiling; teach English clause patterns.
```

---

## OUTPUT STRUCTURE (per learner)
```
1. WRITING PROFILE: target exam, CEFR level, current band/grade estimate from baseline.
2. MARKED SAMPLE: correction codes on the script + per-criterion score + the single
   band-limiting issue named per criterion.
3. TARGETS: exactly one actionable target per criterion.
4. MODEL: one teacher-written exemplar paragraph for the same prompt (annotated with the moves).
5. REDRAFT TASK: rewrite addressing targets, same prompt or a parallel one.
6. (Optional) MICRO-LESSON: one 30–45-min lesson on the weakest criterion.
```

## FAILURE MODES
- Teaching generic "good writing" instead of the learner's actual rubric → fix in Module 1–2.
- Correcting every error → demoralises, no learning; prioritise the band-limiting error.
- Praising language while ignoring an off-topic answer → the score is already capped.
- Reproducing copyrighted descriptor tables → paraphrase criteria; link the official PDF.
- Building the 4-paragraph essay with a sub-B1 learner → premature; build sentence/paragraph first.
