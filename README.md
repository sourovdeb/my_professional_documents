# my_professional_documents — Sourov Deb

The single home for my professional life: my story, my evidence, my tools, and the
engine that turns writing into income and contacts. Built to run on my worst day, not
just my best — every system here assumes uneven energy (bipolar I, ADHD, insomnia) and
carries the day when I can't.

## Map of the repository

| Folder | What lives here |
|---|---|
| **GROWTH/** | The active engine. How I get exposure by writing, the free tools I use, the job/partner automation, and the self-care that keeps it running. **Start here.** |
| **Story_of_Sourov/** | Canonical archive of my life story, CV, medical, regulatory and reusable skills. The source of truth. |
| **Biography_and_Medical/** | Plain-language versions of my biography and medical records. |
| **Legal_Documents/** | Cambridge / CELTA case file (French). |
| **CELTA_Teaching_Materials/** | Cambridge teaching frameworks and guidance notes. |
| **Profile_Documents/** | CV models, motivation letters, the certified-trainer one-pager. |
| **Communications/** | Published posts (Substack, Medium, LinkedIn) and correspondence. |
| **Email_Extension/** | My open-source Chrome extension for AI-assisted Gmail draft automation. |
| **_archive/** | Old raw `.zip` exports and duplicate extractions. Nothing unique — kept only as backup. |

## How to use this repo

1. Every morning, open **GROWTH/README.md** and do the one thing it tells you.
2. Write the 500 words. Publish. Log it.
3. Let the automation surface jobs and like-minded people while I write.
4. On a crash day, follow the four anchors in `Story_of_Sourov` / Daily Living Strategy. That still counts.

Quality over quantity. Active voice. One real thing a day.
